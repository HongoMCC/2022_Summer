## Windowsとは?
そもそも""Windowsって何？""というお話です。
**Mirosoft Windows**(マイクロソフト ウィンドウズ)は、Microsoft社製のPC向け汎用OS(オペレーティングシステム)シリーズです。  
OSとは、PCのハードの違いをソフトで吸収し、応用ソフトの実行環境を提供するソフトです。  
現在では、Windows 10が主流で、後継のWindows 11も普及し始めています。
